Detailed Description

Represents a gamer's detailed standing on a leaderboard.

Member Function Documentation

bool GameUp.Rank.isNew ( )
Returns
true if this is the first time the current gamer appears on this leaderboard, false otherwise.
bool GameUp.Rank.isNewBestRank ( )
Returns
true if this response contains a new all-time best rank on this leaderboard, false otherwise.
bool GameUp.Rank.isNewRank ( )
Returns
true if the rank has changed since it was last checked, regardless if it's now higher or lower, false otherwise.
bool GameUp.Rank.isNewScore ( )
Returns
true if the response indicates the gamer has a new best score on this leaderboard, false otherwise.
