Detailed Description

Represents an achievement's data, also contains gamer progress if applicable.

Member Enumeration Documentation

enum GameUp.Achievement.AchievementState
strong

Achievement State, referring to how display should be handled.

Enumerator
VISIBLE 

The achievement and all its details are available to the gamer.

SECRET 

The name and description are replaced with "???", unless the gamer has completed the achievement.

HIDDEN 

The achievement does not appear at all, unless the gamer has completed the achievement.

enum GameUp.Achievement.AchievementType
strong

Achievement Type, referring to how gamer interaction must occur.

Enumerator
NORMAL 

Standard earned/unearned achievement.

INCREMENTAL 

Incremental achievement, requiring some number of actions before it is awarded, subject to game logic.
